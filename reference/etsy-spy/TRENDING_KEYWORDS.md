## 🛠️ Research Sources

### Free Tools
1. **Etsy Search Autocomplete** - Type keyword, see suggestions
2. **eRank** (free tier) - Etsy keyword trends
3. **Google Trends** - Compare keyword popularity
4. **Pinterest Trends** - Visual trend discovery

### Paid Tools
1. **Marmalead** - Etsy SEO/keywords
2. **Sale Samurai** - Etsy analytics
3. **eRank Pro** - Advanced Etsy research

### Manual Research
1. **Etsy Bestsellers** - `/c/clothing?explicit=1&order=most_relevant`
2. **Etsy Gift Guides** - Seasonal trending
3. **TikTok/Instagram** - Viral trends
4. **Reddit** - r/Etsy, r/EtsySellers
